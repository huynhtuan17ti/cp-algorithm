## Problem
Given number $N$, could you define the number of trees has $N$ vertices.

## Solution
The number of tree has $N$ vertices is calculated by $N^{N-2}$

| n-vertex | answer |
|----------|--------|
| 1        | 1      |
| 2        | 1      |
| 3        | 3      |
| 4        | 16     |
| 5        | 125    |
| 6        | 1296   |
